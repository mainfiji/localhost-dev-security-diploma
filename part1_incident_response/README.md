## Ключевые результаты

- Восстановлена связность между офисами (исправлен Endpoint в WireGuard)
- Исправлены избыточные правила МСЭ (SSH/RDP доступ только из сегмента управления)
- Настроен прозрачный прокси (PolarProxy), обнаружена передача персональных данных в LLM (Qwen)
- Обнаружены 2 теневых хоста (phpLDAPadmin и пользовательская ВМ)
- Подобраны 25 слабых паролей из 68 учётных записей
- Выявлены избыточные права администраторов (adm_rsafin в T0 группе)
- Восстановлены права на файловом сервере, удалены нелегитимные учётные записи
- Расследованы 3 инцидента (SQLi + утечка CRM, майнер xmrig, фишинг почты)
- Проведён аудит по ФСТЭК №21: выявлены несоответствия в 8 требованиях

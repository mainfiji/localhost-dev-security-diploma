## Описание задачи

Настроить прозрачный прокси для инспекции исходящего TLS-трафика пользователей с помощью PolarProxy и Arkime. Проанализировать трафик на предмет передачи персональных данных в LLM-сервисы.

## Настройка DNAT на srv_fw

```bash
sudo nft add rule ip nat PREROUTING ip saddr 192.168.202.0/24 tcp dport 443 ip daddr != {192.168.0.0/16, 10.97.0.0/16} dnat to 192.168.204.204:10443
```

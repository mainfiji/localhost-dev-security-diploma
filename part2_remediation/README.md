# Обновления корпоративных политик и стандартов

В этой папке собраны результаты анализа системных проблем информационной безопасности, а также выполненные настройки для устранения несоответствий требованиям регулятора (ФСТЭК №21).

## Содержание

- [Анализ системных проблем](system_analysis.md) — перечень проблем, их источники и пути решения.
- [Устранение отклонений от требований регулятора](compliance_fixes.md) — выполненные настройки на рабочих станциях, сервере LDAP и результаты проверок.

## Связь с другими разделами

- Настройки, описанные в `compliance_fixes.md`, реализованы в инфраструктуре и проверены через Ansible.
- Анализ системных проблем послужил основой для доработки корпоративных стандартов (Политика управления учётными записями и Инструкция по безопасной настройке ОС Linux).

Все изменения задокументированы и могут быть использованы для подготовки к проверке регулятора.

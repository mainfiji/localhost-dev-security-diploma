## Групповые политики (GPO)

### Политика паролей

| Параметр | Значение |
|----------|----------|
| Минимальная длина | 8 символов |
| Сложность | Буквы разных регистров, цифры, спецсимволы |
| Срок действия | 180 дней |
| История | 3 пароля |
| Блокировка после | 5 неудачных попыток |
| Сброс блокировки через | 15 минут |

### Политика доступа

- Гостевая учётная запись отключена
- Локальный пользователь Guest отключён
- Интерактивный вход для локальных пользователей запрещён (кроме администраторов)
- Доступ по RDP только для групп `Domain Admins` и `RDP_Users`

### Политика аудита

| Событие | Включено |
|---------|----------|
| Входы и выходы |  |
| Управление учётными записями |  |
| Изменения политик и прав |  |
| Доступ к объектам файловой системы |  (для общих папок) |

## Сетевые потоки для Active Directory

| Протокол | Порт | Назначение | Источник → Назначение |
|----------|------|------------|----------------------|
| LDAP | 389 | Аутентификация | Все сегменты → DC |
| LDAPS | 636 | Аутентификация (SSL) | Все сегменты → DC |
| Kerberos | 88 | Аутентификация | Все сегменты → DC |
| DNS | 53 | Разрешение имён | Все сегменты → DC |
| SMB | 445 | Общие ресурсы | Все сегменты → DC |
| RPC | 135 | Управление | Сегмент управления → DC |
| NTP | 123 | Синхронизация времени | DC → srv_ntp |

---

## Проверка применения политик

- Клиентская ВМ успешно присоединена к домену.
- При входе под доменным пользователем применяются все настроенные политики.
- Локальные настройки (например, парольная политика) заблокированы и не редактируются.

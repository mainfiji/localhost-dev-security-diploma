# Сетевые потоки для Active Directory

## Таблица портов и протоколов

| Протокол | Порт | Назначение | Источник → Назначение |
|----------|------|------------|----------------------|
| LDAP | 389/TCP | Аутентификация и запросы к каталогу | Все сегменты → Контроллер домена |
| LDAPS | 636/TCP | Аутентификация (SSL) | Все сегменты → Контроллер домена |
| Kerberos | 88/TCP, UDP | Аутентификация | Все сегменты → Контроллер домена |
| DNS | 53/UDP, TCP | Разрешение имён и поиск служб AD | Все сегменты → Контроллер домена |
| SMB | 445/TCP | Общие ресурсы (SYSVOL, NETLOGON) | Все сегменты → Контроллер домена |
| RPC | 135/TCP | Удалённый вызов процедур (службы AD) | Сегмент управления → Контроллер домена |
| NetBIOS | 137-139/UDP, TCP | Служба имён NetBIOS (опционально) | Все сегменты → Контроллер домена |
| RDP | 3389/TCP | Удалённое администрирование | Сегмент управления → Контроллер домена |
| Динамические RPC | 49152-65535/TCP | Различные службы AD | Сегмент управления → Контроллер домена |
| NTP | 123/UDP | Синхронизация времени | Контроллер домена → srv_ntp (192.168.201.123) |

## Схема взаимодействия
[Все сегменты] ──(LDAP, Kerberos, DNS, SMB)──→ [Контроллер домена (int.local.host)]
[Сегмент управления] ──(RPC, RDP)──→ [Контроллер домена]
[Контроллер домена] ──(NTP)──→ [srv_ntp (192.168.201.123)]

text

Все потоки разрешены только по принципу минимальной необходимости (least privilege).
